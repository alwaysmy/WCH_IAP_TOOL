实现一个类似于WCH这个官方IAP WCHMcuIAP_WinAPP.exe，根据其下位机功能和使用说明，写一个我自己用的上位机，下载接口只保留USB,具有设备列表选择和搜索设备按钮，有个下载栏，可以填入地址，也可以把文件拖进去，旁边有个下载文件选择按钮，下面有个程序下载按钮，最下面是日志区，用C#编写，本机上有dotnet环境。

USB接口可阅读单片机程序，USB驱动调用my_wch_iap_app文件夹内的CH375库（这个接口用的就是CH375的驱动）
