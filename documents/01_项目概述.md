# 项目概述

## 1. 项目背景

本项目是一个用于沁恒（WCH）单片机IAP（In-Application Programming）下载的上位机工具。参考官方工具 `WCHMcuIAP_WinAPP.exe` 实现，提供CLI和GUI两个版本。

## 2. 功能概述

### 2.1 核心功能

| 功能 | 说明 |
|------|------|
| USB设备搜索 | 自动检测连接的WCH IAP设备 |
| HEX转BIN | 支持Intel HEX格式转换为BIN格式 |
| 程序下载 | 通过USB接口下载程序到单片机 |
| Flash擦除 | 下载前擦除Flash区域 |
| 数据校验 | 下载后验证数据正确性 |

### 2.2 支持的文件格式

- **BIN文件**：直接下载
- **HEX文件**：自动转换为BIN后下载

### 2.3 版本对比

| 特性 | CLI版本 | GUI版本 |
|------|---------|---------|
| 界面 | 命令行 | 图形界面 |
| 设备选择 | 自动选择第一个 | 列表选择 |
| 文件选择 | 命令行参数 | 对话框/拖放 |
| 日志输出 | 控制台 | 文本框 |
| USB自动检测 | 无 | 支持 |

## 3. 技术架构

### 3.1 开发环境

- **语言**：C# (.NET 9.0)
- **框架**：Windows Forms (GUI版本)
- **IDE**：Visual Studio Code / Trae IDE

### 3.2 依赖库

- **CH375DLL64.dll**：USB驱动库，用于与CH375芯片通信

### 3.3 IAP协议

基于单片机固件定义的IAP协议：

| 命令 | 值 | 说明 |
|------|-----|------|
| CMD_IAP_PROM | 0x80 | 编程命令 |
| CMD_IAP_ERASE | 0x81 | 擦除命令 |
| CMD_IAP_VERIFY | 0x82 | 校验命令 |
| CMD_IAP_END | 0x83 | 结束命令 |
| CMD_JUMP_IAP | 0x84 | 跳转命令 |

### 3.4 响应码

| 响应 | 值 | 说明 |
|------|-----|------|
| ERR_SUCCESS | 0x00 | 成功 |
| ERR_ERROR | 0x01 | 错误 |
| ERR_End | 0x02 | 结束（正常） |

## 4. 下载流程

```
1. 读取下载文件内容
   └─ HEX格式转换成BIN格式
   
2. 打开下载接口
   └─ 通过CH375 DLL打开USB设备
   
3. 擦除FLASH
   └─ 发送CMD_IAP_ERASE命令
   
4. 写FLASH
   └─ 分块发送CMD_IAP_PROM命令
   
5. Flash检验
   └─ 发送CMD_IAP_VERIFY命令
   
6. 下载结束
   └─ 发送CMD_IAP_END命令
   └─ 单片机跳转到APP
   
7. 关闭下载接口
```

## 5. 目标设备

- **单片机型号**：CH32V305GBU6
- **USB VID**：0x4348
- **USB PID**：0x55E0
- **Flash基地址**：0x08005000
